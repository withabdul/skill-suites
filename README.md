# Skill Suites

A collection of BMad skill modules for AI-powered workflows.

## Modules

### 🎓 Courses & Bootcamp Suite (`crs`)

End-to-end course creation ecosystem built on PDLC (Product Development Lifecycle). Supports hybrid online/offline delivery with 9 specialized agents.

**Agents:**
- `crs-agent-manager` — Orchestrator, manages full PDLC flow
- `crs-agent-consultant` — Discovery & needs analysis
- `crs-agent-theory-writer` — Curriculum & content writing
- `crs-agent-architect` — Course structure & timeline design
- `crs-agent-code-smith` — Code examples & exercises
- `crs-agent-validator` — Quality assurance & review
- `crs-agent-delivery` — Slide & delivery format production
- `crs-agent-grader` — Assessment & automated grading
- `crs-agent-student-sim` — Student simulation & feedback

## Installation

```bash
npx skills@latest install withabdul/skill-suites
```
